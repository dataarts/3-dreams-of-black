Take a look behind the curtain

"3 Dreams of Black", an interactive film by Chris Milk, made with some friends at Google, was produced in part to showcase the creative potential of WebGL.  WebGL is a context of the HTML5 canvas element that enables hardware-accelerated 3D graphics in the web browser without a plug-in. In other words, it enables your browser to do some really cool visual stuff. On this page, you’ll find a WebGL model viewer featuring several of the "3 Dreams of Black" creatures. You’ll also find 8 demos created by our development team to help you get started with WebGL, a link to the code base, and links to demos created with Three.js, the JavaScript library that powers much of the
"3 Dreams of Black" experience

Ready to get the entire code base?

"3 Dreams of Black" is entirely open source, and the project is hosted on GitHub, a popular social coding site. It's a bit of a mess, so proceed at your own risk, but if you poke around enough, you’ll see how we created the entire project. Fork away.
